Selecciona un método

Realiza entradas de datos con potencias usando ^ (ej. x^2).

Separación

Ingresa ecu en: dy/dx = ecu

Ejemplos: y, x*y, x*sen(y)

Exactas

Ingresa M y N en: M dx + N dy = 0

Ejemplos: M=2*x*y, N=x^2

Consejos

  • Usa ^ para potencias.
  • Forma explícita suele ser más legible.
  • Para Euler usa exp(x)
Entrada: usa ^ para potencias (recomendado). También se acepta **.
